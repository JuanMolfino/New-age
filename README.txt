# ⚔️ New Age Dungeon

Un simulador de combate por turnos hecho en **JavaScript**, **HTML** y **CSS**.  
Elige a tu héroe, enfrenta a un monstruo y mira cómo se desarrolla la batalla con barras de vida y animaciones simples.

## 🚀 Cómo jugar
1. Ingresa tu nombre y guárdalo.
2. Selecciona tu personaje y el monstruo que enfrentarás.
3. Haz clic en **"¡Comienza la aventura!"**.
4. Observa cómo se desarrolla la pelea ronda a ronda.

## 🛠️ Tecnologías utilizadas
- **JavaScript** para la lógica del juego.
- **HTML5** para la estructura.
- **CSS3 / SCSS** para estilos y animaciones.
- **SweetAlert2** para mensajes de victoria, derrota o empate.

## 📂 Estructura del proyecto
📦 NewAgeDungeon
┣ 📜 index.html # Estructura base del juego
┣ 📜 main.js # Lógica principal de combate
┣ 📜 style.css # Estilos y animaciones
┣ 📜 data.json # Datos de personajes y monstruos
┣ 📂 media # Imágenes y recursos gráficos
## ✨ Características
- Combate por turnos con daño físico y mágico.
- Animaciones básicas de ataque.
- Barras de vida que se actualizan en tiempo real.
- Guardado de nombre en **LocalStorage**.
- Datos de personajes y monstruos cargados desde **JSON** con `fetch()`.
